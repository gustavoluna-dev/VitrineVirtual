# Vitrine Virtual - Painel Administrativo (Admin Panel)

Esta pasta contém o Painel Administrativo do projeto **Vitrine Virtual**, onde é possível gerenciar produtos, categorias, usuários e configurações gerais da vitrine.

Para instruções de configuração, dependências e inicialização completa do projeto, consulte o [README principal da raiz do projeto](../../README.md).

## Tecnologias principais utilizadas aqui:
*   React 19
*   Vite 8
*   Tailwind CSS v4 (com `@tailwindcss/vite`)
*   Lucide React
